# Have-To-Do

### 간단 설명
- React Native Expo로 만든 TodoList 관리 앱입니다.
- Android만 지원합니다.

### 제공 기능
- 해야할 일 리스트를 작성할 수 있습니다.
- 작업 완료를 표시할 수 있습니다.
- Todo 항목을 수정하고 삭제할 수 있습니다.
- AsyncStorage를 사용해 Todo 항목을 저장합니다.
